# Odin Recipes

## Description
This is a basic recipe website built as part of **The Odin Project's Foundations Curriculum**. The goal of this project was to move away from a "guided" environment and start building a real-world project locally on a Linux environment (Xubuntu) using a professional developer workflow.

## Skills Demonstrated
Upon completion of this project, I have demonstrated the following skills:

*   **HTML Structure:** Proper use of boilerplate, headings, paragraphs, and links.
*   **Data Organization:** Utilizing unordered (`<ul>`) and ordered (`<ol>`) lists to represent ingredients and preparation steps.
*   **Relative File Paths:** Navigating local directories to link multiple HTML pages and assets.
*   **Git Workflow:** Using the terminal to stage (`git add`) and commit (`git commit`) logical units of work following the "7 Rules of Git Commits."
*   **Development Environment:** Working within a Linux VM, managing files via the CLI, and utilizing VS Code extensions for efficiency.

## Technologies Used
*   HTML5
*   Git & GitHub
*   Linux (Xubuntu)
